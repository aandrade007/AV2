# Aerocode - AV2
 
Sistema de gerenciamento de produção aeroespacial focado no controle de aeronaves, peças, etapas e funcionários.
 
---
 
## Requisitos
 
* Windows 10 ou superior / Linux Ubuntu 24.04.03 (ou derivadas).
* Node.js instalado.
---
 
## Como Rodar o Projeto
 
**1. Clone o repositório e acesse a pasta:**
 
```bash
git clone https://github.com/aandrade007/AV2.git
cd AV2/aerocode
```
 
**2. Instale as dependências:**
 
```bash
npm install
```
 
**3. Inicie o servidor local:**
 
```bash
npm run dev
```
 
**4. Acesse no navegador:**
 
Abra [http://localhost:3000](http://localhost:3000)
